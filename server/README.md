# GoLang Server

# Usage

## Start as dev

`make start`

## Build for VPS

`make build-server`

## Build for Shared Hosting

`make build-server-fcgi`

## Run tests

`make test`

## Run docker

This is to create a test environment
It includes handy environments to test:
- database queries
- email smtp test server (TODO)

`make docker`

# Dependencies

Please install docker, golang, vscode-thunderclient
